# Accessibility Checklist (Navigation & Search)

## Keyboard and Focus
- [ ] All interactive elements reachable via keyboard
- [ ] Visible focus indicator on menu and search elements
- [ ] No keyboard traps

## Structure and Semantics
- [ ] Navigation is grouped logically
- [ ] Heading structure supports page scanning
- [ ] Interactive elements use appropriate roles

## Non-Text Content
- [ ] Icons used in navigation have accessible labels
- [ ] Decorative images are properly marked

## Visual Design
- [ ] Sufficient color contrast for text and controls
- [ ] Readable typography and spacing
