# Accessibility Checklist (Quick Audit)

## Keyboard & Focus
- [ ] Can navigate all interactive elements via keyboard
- [ ] Visible focus indicator
- [ ] No keyboard trap

## Structure
- [ ] One H1 per page (reasonable)
- [ ] Heading order logical (H2 under H1, etc.)
- [ ] Lists/tables used semantically

## Non-text Content
- [ ] Images have appropriate alt text (or decorative)
- [ ] Icons have labels/aria where needed

## Contrast & Typography
- [ ] Sufficient color contrast for text
- [ ] Readable font choice (sans-serif acceptable)
