# Test Scope

## In Scope
This QA case study focuses on evaluating the usability and accessibility of core information access points on the CIDI website.

- Global navigation menu (top navigation bar)
- Dropdown menu structure and interactions
- Search entry point and search result behavior
- Keyboard accessibility for navigation and search components
- Visual and structural accessibility related to navigation and search

## Out of Scope
The following areas are intentionally excluded to maintain a focused and realistic QA scope:

- Backend logic, search indexing, or relevance algorithms
- Authentication-protected service portals
- Performance, load, or security testing
- Full WCAG conformance audit across all pages

## Test Environment
- Desktop:
  - macOS
  - Browsers: Chrome, Safari
- Mobile (exploratory, optional):
  - iOS Safari

## Assumptions and Constraints
- Website content and structure may change over time
- Findings represent the state of the site at the time of testing
- Testing is observational and documentation-based, with no access to internal code or systems
